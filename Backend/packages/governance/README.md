# arkon-governance

Python package: same agent logic as `new.py`, structured for FastAPI (no Rich terminal).

Install (editable, from repo root):

```bash
pip install -e ./packages/governance
```

Requires `AZURE_OPENAI_API_KEY`, `AZURE_OPENAI_ENDPOINT`, `AZURE_OPENAI_CHAT_DEPLOYMENT` (optional `AZURE_OPENAI_REASONING_DEPLOYMENT`).
