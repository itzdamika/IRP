Arkon — run locally
==================

1) API (terminal A), from repo root:

   pip install -r apps/api/requirements.txt
   pip install -e packages/governance

   cd apps/api
   set PYTHONPATH=.
   set JWT_SECRET=dev-secret-change-me
   uvicorn app.main:app --reload --host 0.0.0.0 --port 8000

2) Web (terminal B):

   cd apps/web
   copy .env.example .env.local
   npm run dev

3) Open http://localhost:3000 — sign up, then chat.

Data: SQLite file at repo ./data/arkon.db (created on first start).
